# Iris Flower Classification

## About the Project
This project classifies Iris flowers into three species (Setosa, Versicolor, Virginica) using Machine Learning.

## Internship
Oasis Infobyte Data Science Internship

## Dataset
Built-in Iris dataset from scikit-learn library (150 samples, 4 features)

## Libraries Used
- Python 3.11.9
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model Used
Random Forest Classifier

## Model Accuracy
100%
